2b. Color Justification

I chose the color red for a GPA that was the equivalent letter grade of an F, orange for a D, yellow for a C, pale green for a B, and dark green for an A. I believe these colors represent a negative to positive gradient that correlates with whatever GPA each student obtained. In general, colors with red or orange hues represent something negative, yellow is neutral, and green represents something positive. These colors accurately portray the sentiment behind GPA numbers.

It would be ideal if I could have coded a gradient for each GPA range. For example, majority of the GPA's fall within the B range, which is the pale green color. Ideally, I would have coded a gradient within each letter grade range excluding an A. Where any GPA in the B range would start at the lightest hue of pale green for a B- and blend into the darkest hue of pale green for a B+, while still making it distinguishable from the A color of dark green. I would have done this for each GPA range, so the spectrum of colors made the grades more distinguishable and significant.
